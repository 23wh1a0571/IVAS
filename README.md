# 🌐 IVAS — Intelligent Voyage Assistance System

## 📌 Overview
IVAS (Intelligent Voyage Assistance System) is an AI-powered travel planning web application that helps users create personalized travel itineraries with advanced features like budget tracking, visa information, currency conversion, and multilingual support.

It combines a modern frontend UI with a Node.js + MongoDB backend for authentication and user management.

---

## 🚀 Features

### ✈️ Travel Planning
- Smart AI-generated itineraries  
- Day-wise travel plans  
- Weather insights and travel suggestions  

### 💰 Budget Management
- Daily expense tracking  
- Budget estimation & remaining balance  
- Expense categorization  

### 🌍 Multilingual Support
- Supports 50+ languages  
- Dynamic UI translation  

### 💱 Currency Converter
- Real-time conversion  
- Multiple currency comparisons  

### 🛂 Visa & Safety Info
- Visa requirements  
- Safety index and travel guidelines  

### 🏨 Travel Services
- Hotel recommendations  
- Flight booking suggestions  

### 🗣️ Phrasebook
- Local language phrases  
- Pronunciation guidance  

### 🌿 Sustainability
- Carbon footprint estimation  

### 🤖 AI Chatbot
- Travel assistant for quick queries
